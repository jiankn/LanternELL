## 二、美国纸张标准

> **美国标准纸张是 US Letter (8.5" × 11")，不是 A4！**

| 纸张 | 尺寸 | 用于 |
|------|------|------|
| **US Letter** ✅ | 8.5" × 11" (215.9 × 279.4 mm) | 美国所有学校、家庭的默认打印纸 |
| A4 ❌ | 8.27" × 11.69" (210 × 297 mm) | 欧洲/亚洲标准，美国不使用 |

代码中已正确设置 `format: 'Letter'`。

## 三、图片设计策略（已确认）

### 3.1 认知科学基础

K-2 学生（5-8 岁）的记忆特征：

| 认知特征 | 设计要求 |
|----------|----------|
| 双编码理论 (Dual Coding) | 图 + 文字同时呈现 = 记忆增强 2× |
| 情绪驱动记忆 | 引发正面情绪的图记忆保留高 40% |
| 具象思维阶段 | 需要看到"像真的东西"，抽象符号无效 |
| 视觉注意力有限 | 图片必须简洁、主体突出、背景干净 |
| 颜色敏感期 | 高饱和度暖色系最吸引注意 |
| 重复强化 | 统一风格 → 品牌认知 → 安全感 → 愿意学 |

### 3.2 确认方案：Kawaii 亲切风 + 粗描边混合

```
Kawaii 拟人化物品 + 明确粗轮廓线 + 高饱和色彩 + 白色干净背景
```

| 效果 | 说明 |
|------|------|
| 🎯 小孩喜欢 | kawaii 表情创造情感连接 |
| 🎯 一眼识别 | 物品形状准确，粗轮廓线突出 |
| 🎯 彩色打印 | 鲜艳好看 |
| 🎯 黑白打印 | 粗轮廓线确保无损辨识 |
| 🎯 批量一致 | AI 可稳定生成统一风格 |
| 🎯 品牌辨识 | 在 TPT/Etsy 竞品中一眼认出 LanternELL |

## 四、AI 图片生成提示词（Imagen 4）

### 4.1 按年级段分类的风格提示词

#### K-2（5-8 岁）— Kawaii + 粗描边

```
STYLE_PROMPT_K2 = "kawaii cute style with a happy smiling face, big round eyes, small pink blush cheeks, thick clean black outlines, bright saturated cheerful colors, simple rounded shapes, white clean background, no text, no words, no letters, professional children's educational flashcard illustration, consistent art style"
```

**完整 Prompt 拼接方式：**
```
[物品描述 image_prompt], [STYLE_PROMPT_K2]
```

**示例：**
```
A yellow pencil with a pink eraser on top, kawaii cute style with a happy smiling face, big round eyes, small pink blush cheeks, thick clean black outlines, bright saturated cheerful colors, simple rounded shapes, white clean background, no text, no words, no letters, professional children's educational flashcard illustration, consistent art style
```

#### 3-5（8-11 岁）— 现代扁平矢量 + 微表情

```
STYLE_PROMPT_35 = "modern flat vector illustration, friendly subtle expression, thick clean outlines, vibrant saturated colors, geometric clean shapes, educational material style, white clean background, no text, no words, no letters, professional children's educational flashcard illustration, consistent art style"
```

#### 6-8（11-14 岁）— 精致动漫风 + 轻度写实

```
STYLE_PROMPT_68 = "clean anime illustration style, semi-realistic proportions, cell shaded coloring, vibrant colors, detailed but clean design, thick outlines, white clean background, no text, no words, no letters, professional educational material illustration, consistent art style"
```

### 4.2 Vocabulary Card 图片的 image_prompt 编写规范

每个词汇条目的 `image_prompt` 应遵循：

1. **以 "A single" 开头** — 确保生成单个物品
2. **描述物品的关键特征** — 颜色、形状、典型使用状态
3. **面向美国课堂** — 物品应为美国学校常见的样式
4. **不要写场景** — 只描述物品本身，场景由 style prompt 控制
5. **不要写 "cute" 或 "kawaii"** — 由 style prompt 统一控制

**好的 image_prompt 示例：**
```json
{ "en": "pencil", "image_prompt": "A single yellow pencil with a pink eraser on top" }
{ "en": "scissors", "image_prompt": "A single pair of child-safe scissors with rounded tips and blue handles" }
{ "en": "backpack", "image_prompt": "A single colorful child's school backpack with two straps" }
{ "en": "clock", "image_prompt": "A single round analog wall clock showing 9 o'clock" }
```

**不好的 image_prompt（避免）：**
```json
// ❌ 太模糊
{ "en": "book", "image_prompt": "An open book with colorful illustrations" }
// ❌ 描述场景而非物品
{ "en": "door", "image_prompt": "A classroom door, slightly ajar, showing a hallway" }
// ❌ 多个物品
{ "en": "shelf", "image_prompt": "A wooden classroom shelf with a few books and toys" }
```

**修正后：**
```json
{ "en": "book", "image_prompt": "A single closed hardcover book, red cover" }
{ "en": "door", "image_prompt": "A single classroom door, front view, with a small window and handle" }
{ "en": "shelf", "image_prompt": "A single small wooden bookshelf, empty, three shelves" }
```

### 4.3 Worksheet 图片的 image_prompt 编写规范

Worksheet 中的 AI 图片也使用相同风格，但需根据 worksheet 类型微调：

| Worksheet 类型 | 图片处理 |
|----------------|----------|
| **matching** | 使用与 vocabulary card 相同的 kawaii 图 |
| **tracing** | 使用与 vocabulary card 相同的 kawaii 图（缩小尺寸放在行旁） |
| **coloring** | 使用**线稿版本**（无填色，仅轮廓线） |
| **fill-in-the-blank** | 使用与 vocabulary card 相同的 kawaii 图 |

**Coloring worksheet 专用 style prompt：**
```
STYLE_PROMPT_COLORING = "black and white line art, thick clean outlines, no fill, no shading, no color, kawaii cute style with a happy smiling face, big round eyes, simple rounded shapes, white background, coloring book page for children, no text, no words"
```

### 4.4 AI 图片生成参数

```javascript
// Imagen 4 Fast API 参数
{
    model: 'imagen-4.0-fast-generate-001',
    instances: [{ prompt: fullPrompt }],
    parameters: {
        sampleCount: 1,
        aspectRatio: "1:1"  // 正方形，适合卡片布局
    }
}
```

**图片后处理参数：**
```javascript
// 使用 sharp 压缩
sharp(inputBuffer)
    .resize(512, 512, { fit: 'inside' })
    .jpeg({ quality: 75 })
    .toBuffer()
```

### 9.5 API 对接要点

**不论用哪个生图 API，核心输入都是 `full_prompt` 字段。**

| API 服务 | 适配说明 |
|----------|----------|
| **Midjourney API** | 直接传 `full_prompt`，加 `--ar 1:1 --style raw` |
| **DALL-E 3 (OpenAI)** | `prompt` = `full_prompt`, `size` = "1024x1024" |
| **Stable Diffusion** | `prompt` = `full_prompt`, `negative_prompt` = "text, words, letters, blurry" |
| **Imagen (Google)** | `prompt` = `full_prompt`, `aspectRatio` = "1:1" |
| **Leonardo AI** | `prompt` = `full_prompt`, `width`/`height` = 512 |
| **通义万相 / 可灵** | `prompt` = `full_prompt` 翻译成中文或直接用英文 |

**通用参数建议：**
```json
{
    "size": "1024x1024",
    "aspect_ratio": "1:1",
    "style": "natural / raw",
    "negative_prompt": "text, words, letters, numbers, watermark, blurry, low quality"
}
```

### 9.7 批量调用脚本模板

拿到 API 后，按以下模板编写批量调用脚本：

```javascript
// scripts/batch-generate-images.mjs (模板)
import fs from 'fs';

const batchData = JSON.parse(fs.readFileSync('data/batch-prompts.json', 'utf8'));
const images = batchData.images;

for (let i = 0; i < images.length; i++) {
    const img = images[i];
    
    // 检查是否已生成（跳过已有图片）
    const outputPath = img.output_dir + img.output_filename;
    if (fs.existsSync(outputPath)) {
        console.log(`[SKIP] ${img.id}`);
        continue;
    }

    // 创建目录
    fs.mkdirSync(img.output_dir, { recursive: true });

    // 调用你的 API（替换为实际 API 代码）
    const result = await YOUR_API.generateImage({
        prompt: img.full_prompt,
        size: '1024x1024',
        // ... 其他参数
    });

    // 保存图片
    fs.writeFileSync(outputPath, result.imageBuffer);
    console.log(`[${i+1}/${images.length}] ${img.id}`);

    // 速率控制（根据 API 限制调整）
    await new Promise(r => setTimeout(r, 1000));
}
```
